The positions in the positions.txt file can be converted from OS grid
references to something more sensible using:

http://gridreferencefinder.com/batchConvert/batchConvert.php

Those data are stored in locations.csv.

Should you ever feel the need to write your own conversion utility, there's
some JavaScript which should be relatively easy to convert here:

http://www.movable-type.co.uk/scripts/latlong-gridref-v1.html
